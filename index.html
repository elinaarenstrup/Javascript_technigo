What is JavaScript?




Javascript or JS is used to create and control dynamic website content.

Like animated graphics, slideshows, interactive forms and autocomplete text suggestions.

For example, when Google suggests search terms based on the first letters you have started typing -


THATS JAVASCRIPT!


JavaScript is used to automate processes that users otherwise would do manually.

Like type the whole search word into Google.

JavaScript is used for adding interactivity, it can be used for mobile apps and to play games directly in your browser.

JavaScript commands is typed directly to the HTML document, all major browsers can read JS today.
